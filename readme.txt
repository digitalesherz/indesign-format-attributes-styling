*****"F1_FormatNamenAuflisten_402d.js""F2_FormatAttribute_407d.js""F3_WerteRunden_403d.js"Drei Scripts für CS4, welche erlauben, die Attribute der Absatz- und Zeichenformate aufzulisten und, falls notwendig, gewisse Werte zu runden.Das erste Script setzt man ein, wenn man die Definitionen von mehr als einem (aber nicht von _allen_) Absatz- oder Zeichenformat erfahren will.Das erste Script erzeugt zwei Textrahmen, welche die Namen der Absatz- und Zeichenformate enthalten. Diese Listen können editiert werden (ganze Zeilen entfernen). Dann das Script "F2_FormatAttribute" starten, in den Menüs das Sternchen einstellen. Auf "OK" werden nur die Attribute der Formate in der Liste ausgezogen und in ein neues Dokument eingesetzt.Man kann auch nur das zweite Script starten. Im Dialog hat man vier Möglichkeiten: 1. Das leere Element eingestellt lassen    -> es wird nichts ausgezogen.2. Das Sternchen wählen    -> von sämtlichen Formaten werden die Attribute aufgelistet.3. Einen Formatnamen einstellen   -> nur die Attribute dieses Formates werden ausgezogen.4. Einen Formatgruppe-Namen einstellen   -> die Attribute aller Formate dieser Gruppe werden ausgezogen.Beim Betrachten des Ergebnisses kommt möglicherweise der Wunsch auf, bei gewissen Werten eine "rundere" Zahl zu haben. Bei Dokumenten, welche in QuarkXPress erstellt worden sind, kann zum Beispiel der Wert des automatischen Zeilenabstandes im Dialog mit "120%" eingetragen sein, beträgt aber in in Wirklichkeit "nur" 119.999694824219%.Das dritte Script funktioniert in bezug auf die Auswahl der Formate genau gleich wie das zweite. Mit der aktuellen Version werden die Werte folgender Optionen gerundet:Absatzformate: Schriftgröße, Zeilenabstand, horizontale Skalierung, Grundlinienversatz, Einzug links, Einzug erste Zeile, Einzug rechts, Abstand vor, Abstand nach, Wortabstand, Autom. Zeilenabstand, Tabulatorpositionen, Absatzlinie darüber, Absatzlinie darunterZeichenformate: Schriftgröße, Zeilenabstand, horizontale Skalierung,Grundlinienversatz*****VorbereitungDie Scripts in den Ordner "Scripts Panel" legen, welcher sich im Ordner "Preferences/AdobeInDesign/Version 6.0/Scripts" des eingeloggten Users befindet. Die Namen der Scripts erscheinen in der Palette "Skripten", zu finden unter "Fenster/Automatisierung".*****PrinzipFür die Absatzformate wird ein Dokument erzeugt. Für jedes Format wird eine Seite eingefügt und die Definition in einen zweispaltigen Rahmen eingesetzt.Für die Zeichenformate wird ein separates Dokument erzeugt. Auf jeder Seite zwei Rahmen mit den Definitionen von je einem Format.*****WichtigFür die Formatierung der Definitionen werden im neuen Dokument zwei Absatzformate verwendet. Falls Anpassungen notwendig werden sollten, ist es dadurch einfacher.*****Ausführung1. Das InDesign-Dokument, dessen Format-Attribute aufgelistet    werden sollen, öffnen.2. Die Scripts starten (Doppelklick auf den Namen in der Palette).   Siehe Erklärung oben.*****Rückmeldungen ...... sind sehr willkommen.*****Hilfe bei Problemen ...... sowie weitere Erklärungen liefere ich gerne.*****Einsatz auf eigene GefahrDiese Scripts sind sorgfältig entwickelt und ausgiebig getestet worden. Sie sollten keine Probleme verursachen.
